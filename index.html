주소

EmailJS 발송 설정 (자동 발송에 필요)

emailjs.com에서 무료 계정을 만들고 서비스(SMTP/Gmail 등)와 템플릿을 연결하세요. 템플릿에 {{to_email}}, {{from_email}}, {{subject}}, {{message}} 변수를 넣고, 템플릿의 첨부파일(Attachments) 항목을 동적 변수 {{pdf_attachment}}로 연결하면 PDF가 자동 첨부됩니다. 설정하지 않으면 PDF를 다운로드한 뒤 메일 앱에서 수동으로 첨부하는 방식으로 대체됩니다.

팀 공유 (Firebase) (선택 사항)

firebase.google.com(무료 Spark 플랜)에서 프로젝트를 만들고 Firestore Database를 사용 설정한 뒤, 프로젝트 설정 > 일반 > SDK 설정값의 config 객체를 그대로 붙여넣으세요. 같은 설정값을 팀원 모두가 입력하면 같은 날짜의 체크리스트를 실시간으로 함께 작성/공유할 수 있습니다. 비워두면 이 기기에만 저장됩니다.
